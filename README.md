## Require: 
node ^7.4.0
install via [Brew](http://brew.sh/).
```
brew install node
```

## Setup:
Install JSON Server as a global module
```
npm i json-server -g
```

Install roadhog at ./
```
npm i roadhog -g
```

## Run:
```
npm run server 
```
To start database.

Then open a new terminal
```
npm run backend
```
To start backend.

Enjoy!
